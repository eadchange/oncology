# 抗肿瘤药物临床试验网站 UI/UX 设计文档

## 1. 设计理念

### 1.1 设计原则
- **专业性**: 体现医疗数据平台的专业性和权威性
- **简洁性**: 界面简洁清晰，突出核心功能
- **易用性**: 用户操作简单直观，降低学习成本
- **响应式**: 适配不同设备和屏幕尺寸
- **可访问性**: 符合WCAG 2.1 AA级标准

### 1.2 色彩方案
- **主色调**: 深蓝色 (#1e3a8a) - 体现专业和信任
- **辅助色**: 浅蓝色 (#3b82f6) - 用于链接和按钮
- **强调色**: 橙色 (#f97316) - 用于重要提示和CTA按钮
- **背景色**: 浅灰色 (#f8fafc) - 页面背景
- **文字色**: 深灰色 (#1f2937) - 主要文字
- **次要文字色**: 中灰色 (#6b7280) - 次要信息

### 1.3 字体设计
- **标题字体**: Noto Serif SC (衬线字体，体现专业)
- **正文字体**: Noto Sans SC (无衬线字体，易读性强)
- **代码字体**: JetBrains Mono (等宽字体，用于代码)

### 1.4 视觉语言
- **卡片设计**: 使用阴影和圆角营造层次感
- **图标风格**: 使用Heroicons线性图标
- **按钮样式**: 圆角矩形，悬停效果
- **表单设计**: 清晰的标签和错误提示

#### 3.1.2 Hero区域设计
- **背景**: 渐变色或医疗相关背景图片

- **临床试验数**: 注册的研究总数
